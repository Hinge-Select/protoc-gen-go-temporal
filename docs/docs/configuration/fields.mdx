# Fields

## Options

### cli.aliases

`[]string`

Adds additional aliases to the CLI flag.

```protobuf
message HelloInput {
  string name = 1 [(temporal.v1.field) = {
    cli: {aliases: ["n"]}
  }];
}
```

### cli.name

`string`

Overrides the default CLI flag name.

```protobuf
message HelloInput {
  string name = 1 [(temporal.v1.field) = {
    cli: {name: "username"}
  }];
}
```

### cli.usage

`string`

Overrides the default CLI flag usage.

```protobuf
message HelloInput {
  string name = 1 [(temporal.v1.field) = {
    cli: {usage: "user name"}
  }];
}
```
